# Harbrook

## Chapter 3 state

Harbrook is never seen directly, but Paret's report gives it a clean function. It is the place that bought him out on his last trade run — flour, salt, lamp oil gone, wagon returning empty. In other words, Harbrook is still ordinary enough for commerce. That normality matters because it contrasts with Millhaven's total loss.

Use Harbrook as evidence that the road has not yet recognized its own pattern. Some places are already dead. Some places are still haggling over goods. The plague is moving through a countryside where ordinary trade continues right up against catastrophe.
