# Alder Run

## Chapter 3 state

Alder Run is not visited on-page in Chapter 3, but it matters because it is the first named settlement north of Millhaven. Paret describes it as barely more than six farms and a shrine, which immediately gives it scale: small, exposed, easy to miss, exactly the kind of place a warning might still save if it gets there in time.

Because the POV has not seen it directly yet, keep its profile anchored to what the chapter actually establishes. It is a living rural stop on the threatened line north. It is close enough to Millhaven to matter at once. It is part of the road map Sylvanas is now trying to outrun.
