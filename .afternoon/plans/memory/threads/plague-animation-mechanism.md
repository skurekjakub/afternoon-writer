# Can the partial ritual data become a usable answer before more sites are lost?

This thread remains resolved in theory and newly sharpened in practice. The characters already know how the plague works. Chapter 14 does not reopen that question. Instead, it asks the nastier one: now that they know, where can they still strike before the answer becomes irrelevant?

Jaina's chapter-14 breakthrough is not a new theorem. It is a deployment logic. Granaries, bakehouses, and holding yards become the only remaining places where the outer shell might still be disrupted before it disappears into enough bodies to finish a city from the inside. That is narrower than cure and more useful than abstraction.

So the thread is no longer about discovery. It is about timing, scale, and whether a solved mechanism can still become a meaningful intervention before Stratholme proves the opposite.
