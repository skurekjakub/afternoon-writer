# How far can the network reach to silence compromised witnesses from a distance?

## Chapter 8 state
This thread begins the moment the operative dies mid-sentence. The capture is real. The interrogation is real. The enemy simply reaches farther and faster than the protagonists hoped.

The continuity value here is tactical. Future chapters should remember that the network does not have to send a knife into a room to erase a witness. It can spend a compromised body from elsewhere, and it seems prepared to do so almost immediately.
