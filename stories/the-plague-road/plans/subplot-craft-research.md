# Subplot Craft Notes — The Plague Road

- If it looks detachable, cut it. A real subplot changes the main story when you pull on it.
- The main plot already has three core lines: external plague investigation, Sylvanas/Jaina partnership, and the core character arcs.
- Any extra thread must echo, contradict, or complicate the theme.
- Climax test: it must resolve in the climax, help create it, or be changed by it.

## Keep
- Lor'themar / Silvermoon: institutional echo of isolation vs engagement.
- Mal'Ganis: hidden antagonist threading false help through the investigation.
- Arthas: tragic mirror of conviction without counterweight.

## Do not add
- Separate romance subplot.
- Independent court-intrigue line.
- Stand-alone Jaina backstory track.
- Monster-of-the-week detours.
- Anything that needs a cutaway just to justify itself.

## Practical rule
- In this story, subplots should travel inside the investigation, not beside it.
- Every beat should raise stakes, sharpen theme, or make the climax more inevitable.
